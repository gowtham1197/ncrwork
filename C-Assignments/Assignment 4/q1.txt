float table[2][3] = { { 1.1,1.2,1.3},{2.1,2.2,2.3}};
Answers for the Questions

a) table refers to the base address of 2-dimensional array

b) table+1 refers to the address of second row

c) *(table+1) refers to the address of first column second row

d) (*(table+1)+1) refers to the address of second column second row

e) (*(table)+1) refers to the address of second column first row

f) *(*(table+1) +1) refers to the value of second column second row

g) *(*(table)+1) refers to the value of second column first row

h) *(*(table+1) refers to the value of second row first column

i) *(*(table) + 1)+1 means that 1 is added to the value at second column first row
